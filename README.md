# Itunes Top Albums

The app display a list of top albums from iTunes.

## General info

This is a simple application that display list of the top iTunes albums. One item contains information about the title, author, number on the list and display photo of the album. Each element can switch to a different view by "more info" button. This view shows more details about this album such as price, category and release date.
